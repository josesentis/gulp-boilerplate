# gulp boilerplate

Gulp boilerplate supporting ES6.

## Features

Current features:

Future features:
- SCSS compilation
- SCSS minification
- Font copy to dest folder
- Javascript ES6 transpilation
- Javascript modules concatenation
- SVG Sprite generation
- CRP styles creation
- Image minification
- Watch changes
- Browsersync
- TWIG.JS

## Install

Clone the project and install using yarn or npm

```
yarn
```
```
npm install
```
